# Pipeline
Transfer Learning:
1. Rete pre trained: ResNet / VGG16
2. Freeze layer finali
3. Diamo il dataset e salviamo i vettori di output (embeddings)
4. Addestramento dei seguenti classificatori:
   - SVM
   - KNN
   - MLP
   - Regressore Lineare

5. Metriche:
   - Accuracy
   - F1-score
   - Inference time
   - model size
   - Robustezza
  
6. Creazione dataset esterno:
   - Acquisizone foto
   - Etichettatura tramite ROBOFLOW
   - Passare dati al modello e fare inferenza.
